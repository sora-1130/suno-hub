# J-Suno Tool 🎵

**🚀 [アプリを使ってみる (ここをクリック)](https://j-suno-tool.streamlit.app/)**

Suno AIのための「日本語プロンプト」を自動生成するツールです。
頭の中のイメージを日本語で入力するだけで、Suno AIに最適な「英語プロンプト（Style & Lyrics）」を一瞬で作ります。

## 🚀 機能
- **イメージ変換**: 日本語の抽象的なイメージを、具体的な音楽用語（英語）に変換
- **構成提案**: Intro, Verse, Chorus などの楽曲構成を含めた歌詞ガイドを生成
- **スマホ対応**: モバイルでも見やすいレスポンシブデザイン

## 📦 使い方
1. テキストボックスに「疾走感のあるアニソン」のように入力
2. 「生成」ボタンをクリック
3. 出力されたプロンプトをコピーして [Suno AI](https://suno.com) に貼り付け
